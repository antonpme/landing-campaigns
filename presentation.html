A strategic initiative to modernize our web presence, resolve licensing issues, and enable marketing agility

Why This Redesign Was Necessary

Critical issues that required immediate attention

Font Licensing Violation

Futura PT was hardcoded in hundreds of places. After canceling Adobe Creative Cloud, we received takedown requests from font owners.

Unmaintainable WordPress

Old WordPress installation with legacy code. The original developer left. Changes are slow, risky, and expensive.

Poor Performance

WordPress site loads slowly, affecting user experience and search engine rankings. High bounce rates on mobile.

No Marketing Access

Marketing agency cannot edit content directly. Every change requires developer involvement, slowing down campaigns.

What We Built

A modern, flexible solution that addresses all critical issues

Modern Visual Design

Clean, professional UI that builds trust. Glassmorphism effects, smooth animations, and consistent visual hierarchy.

Mobile-First

Fully responsive design optimized for all devices. Touch-friendly interface captures mobile traffic effectively.

Lightning Fast

Static files load instantly. No database queries, no PHP processing. Better SEO and user experience.

Strapi CMS Integration

Headless CMS for marketing team to manage content directly without developer involvement.

Legal Font Usage

Switched to Jost (Google Fonts) - free, open-source, visually similar. Zero licensing concerns, forever.

Clean Codebase

Single HTML + CSS + JS files. Easy to maintain, update, and deploy. Version controlled on GitHub.

Business Impact

Tangible benefits for Shipa Delivery

Legal Compliance

Zero font licensing risk. No more takedown notices.

Reduced Costs

No WordPress maintenance overhead. Modern static site with Strapi CMS is simpler to host and manage.

Better Conversions

Modern design + fast loading = more leads from the same traffic.

Marketing Agility

Agency can update content directly with Strapi integration.
